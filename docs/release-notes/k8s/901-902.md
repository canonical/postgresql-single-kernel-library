## Highlights

* **Workload upgrade from PostgreSQL 14 to 16**
  * Check the official PostgreSQL [16.0](https://www.postgresql.org/docs/release/16.0/) - [16.13](https://www.postgresql.org/docs/release/16.13/) release notes
* [DPE-6964] **Juju storage pools** for flexible storage configuration options by @marceloneppel in {octicon}`mark-github` [#941](https://github.com/canonical/postgresql-k8s-operator/pull/941)
* [DPE-6189] **Password management with Juju user secrets** by @reneradoi in {octicon}`mark-github` [#926](https://github.com/canonical/postgresql-k8s-operator/pull/926)
* **New in-place upgrades & rollbacks using refresh v3 library** by @marceloneppel in {octicon}`mark-github` [#1115](https://github.com/canonical/postgresql-k8s-operator/pull/1115)
* [DPE-7595] **Switch to TLS v4 library** by @dragomirp in {octicon}`mark-github` [#1116](https://github.com/canonical/postgresql-k8s-operator/pull/1116)
  * New endpoints client-certificates and peer-certificates
  * Endpoint peer-interfaces uses TLS by default
  * See all endpoints on [Charmhub](https://charmhub.io/postgresql-k8s/integrations?channel=16/stable)
* [DPE-7861] Several **improvements to built-in roles** by @marceloneppel in {octicon}`mark-github` [#1049](https://github.com/canonical/postgresql-k8s-operator/pull/1049))
