## Technical notes

* `juju refresh` from the old-stable revision 288 to the current-revision 324 is **NOT** supported!!! The [upgrade](https://charmhub.io/postgresql/docs/h-upgrade) functionality is new and supported for revision 324+ only!
* Please check additionally [the previously posted restrictions](https://charmhub.io/postgresql/docs/r-revision-288).
* Ensure [the charm requirements](https://charmhub.io/postgresql/docs/r-system-requirements) met
* The minimum Juju version required to reliably operate **all** features in this release is `v3.1.5`.

See the [system requirements](https://charmhub.io/postgresql/docs/r-system-requirements) for more details about Juju versions and other software and hardware prerequisites.
