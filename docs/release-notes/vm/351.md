## What's changed

This section outlines the most important highlights, features and bug fixes in this release.

### Highlights

* [Core] Updated `Charmed PostgreSQL` SNAP image ({octicon}`mark-github` [#291](https://github.com/canonical/postgresql-operator/pull/291))([DPE-3039](https://warthogs.atlassian.net/browse/DPE-3039)):
  * `Patroni` updated from 3.0.2 to 3.1.2
  * `Pgbackrest` updated from 2.47 to 2.48
* [Plugins] [Add 24 new plugins/extension](https://charmhub.io/postgresql/docs/r-plugins-extensions) in ({octicon}`mark-github` [#251](https://github.com/canonical/postgresql-operator/pull/251))
  * **NOTE**: extension `plpython3u` is deprecated and will be removed from [list of supported plugins](https://charmhub.io/postgresql/docs/r-plugins-extensions) soon!
* [Config] [Add 29 new configuration options](https://charmhub.io/postgresql/configure) in ({octicon}`mark-github` [#239](https://github.com/canonical/postgresql-operator/pull/239))([DPE-1781](https://warthogs.atlassian.net/browse/DPE-1781))
  * **NOTE:** the config option `profile-limit-memory` is deprecated. Use `profile_limit_memory` (to follow the [naming conventions](https://juju.is/docs/sdk/naming))! ({octicon}`mark-github` [#306](https://github.com/canonical/postgresql-operator/pull/306))([DPE-3096](https://warthogs.atlassian.net/browse/DPE-3096))

## Packaging

* Charmed PostgreSQL ships the latest PostgreSQL “14.9-0ubuntu0.22.04.1”
* PostgreSQL cluster manager Patroni updated to "3.2.1"
* Backup tools pgBackRest updated to "2.48"
* The Prometheus postgres-exporter is "0.12.1-0ubuntu0.22.04.1~ppa1"
* VM charms based on [Charmed PostgreSQL](https://snapcraft.io/charmed-postgresql) SNAP (Ubuntu LTS “22.04” - ubuntu:22.04-based) revision 89
* Principal charms supports the latest LTS series “22.04” only
* Subordinate charms support LTS “22.04” and “20.04” only

## Requirements
The minimum Juju version required to reliably operate **all** features in this release is `v3.1.6+`.

See the [system requirements](https://charmhub.io/postgresql/docs/r-system-requirements) for more details about Juju versions and other software and hardware prerequisites.
