## Known limitations
 * The unit action `resume-upgrade` randomly raises a [harmless error message](https://warthogs.atlassian.net/browse/DPE-5420): `terminated`.
 * The [Sysbench](https://charmhub.io/sysbench) charm may [crash](https://warthogs.atlassian.net/browse/DPE-5436) during a PostgreSQL charm refresh.
 * Make sure that [cluster-cluster replication](https://charmhub.io/postgresql/docs/h-async-set-up) is requested for the same charm/workload revisions. An automated check is [planned](https://warthogs.atlassian.net/browse/DPE-5418).
 * [Contact us](https://charmhub.io/postgresql/docs/r-contacts) to schedule [the cluster-cluster replication](https://charmhub.io/postgresql/docs/h-async-set-up) upgrade with you.

## Requirements
The minimum Juju version required to reliably operate **all** features in this release is `v3.4.3+`.

See the [system requirements](https://charmhub.io/postgresql/docs/r-system-requirements) for more details about Juju versions and other software and hardware prerequisites.
